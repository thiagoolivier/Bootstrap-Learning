Margin and Padding

Margin

In bootstrap, you can use:

  • mt - for margin-top property
  • mr - for margin-right property
  • mb - for margin-bottom property
  • ml - for margin-left property
  • m - if you want to change all four axes

Padding

In bootstrap, you can use:

  • pt - for margin-top property
  • pr - for margin-right property
  • pb - for margin-bottom property
  • pl - for margin-left property
  • p - if you want to change all four axes
